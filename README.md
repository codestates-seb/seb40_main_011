## 💡 서비스 소개
이미 널리 알려진 테크기기의 정보나 후기를 보는 웹사이트는 많이 있습니다만, 정말 많은 정보들이 있어 내가 알고싶은 정보만 찾기가 까다로웠습니다. 
<br>컴팩트한 정보로 보다 좋은 UX를 제공하고자 코드테크 프로젝트를 시작하게 됐습니다.
